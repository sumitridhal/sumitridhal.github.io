The still reference is a **column field**: equal-width bars, hard vertical cuts, and a shared palette that runs black → saturated red → pink → warm off-white toward the bottom. What makes it feel designed instead of a flat gradient is the **per-column threshold**: the centre column holds black the longest; toward the edges the coloured band “starts” higher, so the top of the colour reads like a shallow **chevron** or inverted roofline.

In a fragment shader you never draw rectangles explicitly. You partition horizontal UV into column indices, compute a scalar `d` = distance from the horizontal centre in column space, and map that to a cutoff `th` on the vertical axis. Pixels above `th` stay in ink black; pixels below pick up a **smooth horizontal feather** so the transition is poster-soft instead of aliased. The vertical colour story is just a second ramp driven by how far below the cutoff you are.

```glsl
float cols = 11.0;
float colCenter = (cols - 1.0) * 0.5;
float colId = floor(q.x * cols);
float d = abs(colId - colCenter) / colCenter;
float th = mix(0.34, 0.91, pow(d, 1.08));
float distBelow = th - q.y;
float feather = smoothstep(-0.02, 0.11, distBelow);
```

Motion is cheap: add a low-frequency sine to `th` for the whole **V to breathe**, and a second sine with phase per `colId` so bars drift slightly out of sync (equaliser energy without audio). A high-frequency **hash** on `floor(uv * resolution)` tints luminance for grain; rehashing with `floor(time)` gives a subtle **flicker**. A tight Gaussian bump on the red band approximates **bloom** in one additive line—enough for thumbnails, not a full HDR pipeline.

**Why one pass?** Because every pixel is independent: there is no overdraw, no atlas unwrap, and no sync between CSS columns and WebGL unless you want it. The trade-off is resolution dependence for the grain (tied to `u_resolution`) and the fact that “bars” are mathematical strips—zoomed enormously you will not get subpixel typography unless you add derivatives-aware AA or multisampling.

**Extensions that stay in 2D:** rotate the chevron with a skew on `q` before column quantisation; replace `pow(d, …)` with smooth noise for a wobbly roofline; modulate hue with `atan` of a second field for a thermograph look; or render this pass into a `WebGLRenderTarget` and feed it as a texture into typography or post stack.

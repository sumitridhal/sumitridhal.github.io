Three parallel ticker rows on a flat field—white lowercase type, small circular portraits, sometimes a hairline divider—read as **one motion system**, not three independent sliders. The trick is mechanical: each row is a viewport with `overflow: hidden`, and the moving layer is **exactly two copies** of the same horizontal strip. When the strip has translated by half of its own scroll width, the second copy lines up where the first started, so you can repeat forever with `ease: 'none'` and no visible jump.

## Structure

1. **Row shell** — fixed height, full width, `overflow-x: hidden` (and often `overflow-y: hidden` so subpixel drift does not flash edges).
2. **Track** — `display: inline-flex` (or flex with `width: max-content`) holding **group A** and **group B** with identical children in the same order.
3. **Tween** — animate the track’s `x` (transform) from `0` to `-half` where `half = track.scrollWidth / 2`. GSAP’s `repeat: -1` rewinds the tween to the start values each cycle, which lands on the same pixel alignment as long as both groups match.

```ts
const half = track.scrollWidth / 2
gsap.fromTo(track, { x: 0 }, { x: -half, duration: 24, ease: 'none', repeat: -1 })
```

## Opposite directions

To make the middle row drift the other way, start from the aligned “second copy” pose and tween back toward `0`:

```ts
gsap.fromTo(track, { x: -half }, { x: 0, duration: 32, ease: 'none', repeat: -1 })
```

Visually the glyphs slide the opposite direction while the seam logic is unchanged.

## Timing and parallax

Stagger **duration** per row (and optionally add a tiny `padding-left` offset on the middle track) so the composition feels less grid-like. Keep every row on linear easing; easing in the middle of a seamless loop introduces a micro-pause at the wrap unless you build a custom wrap-aware curve.

## Reduced motion

When `prefers-reduced-motion: reduce` is set, stop the tweens and park each track at a fixed `x` so the layout still reads as the same hero, without continuous drift—matching how other writing embeds on this site treat optional motion.

## What to steal first

- **Half-width loop invariant** — measure once per layout pass; if fonts or copy change, `invalidateOnRefresh` on a `ScrollTrigger` wrapper (or a `ResizeObserver` that re-measures) keeps the seam tight.
- **Linear infinite tweens** — reserve easing for one-shot entrances; marquees stay `ease: 'none'`.
- **Row independence** — duplicate *per row* if the sequences differ (as in the preview); do not share one global strip unless every row’s content matches.

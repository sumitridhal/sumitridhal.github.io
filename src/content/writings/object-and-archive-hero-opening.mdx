## Part 1 ‚Äî Hero curtain and split reveal

**Target:** A full-bleed **curtain** over a split hero; **driver** is first paint (then optional replay); **runtime** is **CSS** only. `WritingObjectArchiveHeroRevealPreview` toggles classes (`is-lifting` ‚Üí `is-open`) so the curtain runs `writing-oa-hero-curtain-up` once; reduced motion snaps the curtain off and disables replay.

Curtain rules and keyframes live under `.writing-oa-hero*` in `_writing.scss`:

```scss
.writing-page__body .writing-oa-hero__curtain.is-lifting {
  animation: writing-oa-hero-curtain-up 1.28s cubic-bezier(0.33, 0, 0.2, 1) forwards;
}

.writing-page__body .writing-oa-hero__curtain.is-open {
  transform: translate3d(0, -100%, 0);
}

@keyframes writing-oa-hero-curtain-up {
  from {
    transform: translate3d(0, 0, 0);
  }
  to {
    transform: translate3d(0, -100%, 0);
  }
}
```

Stack (bottom ‚Üí top): split halves, curtain, wordmark + scrim.

## Part 2 ‚Äî Left column: stacked plates and scale ladder

**Target:** Sixteen centered `<img>` plates, **uniform `scale` only**. **Driver:** **Next ‚Üí** / **‚Üê Back** stepping `advanceCount`. **Runtime:** one **GSAP `timeline`** per step; every layer tweens in **parallel** at label time `0` (`power2.inOut`, duration `0` when reduced motion).

Core pattern in `WritingObjectArchiveLeftPanelPreview`:

```ts
const tl = gsap.timeline({
  defaults: { duration: dur, ease: EASE },
  onComplete: () => {
    if (alive) setIsAnimating(false)
  },
})

imgs.forEach((img, j) => {
  tl.to(img, { scale: targetScale(j, advanceCount, n) }, 0)
})
```

`targetScale` maps each index to **passed** (full-bleed), **focal** (mid), or **ahead** (hidden) scale constants.

## Part 3 ‚Äî Composite: static Part 1 hero + ‚ÄúCurrent obsessions‚Äù scroll strip

**Target:** CSS variable **`--oa-cover-p`** in `0‚Ä¶1` over one viewport of vertical travel; obsessions block **`translateY`** and intro **`padding-top`** follow it. **Driver:** wheel on the shell (smooth path) or native scroll (reduced). **Runtime:** either **Locomotive‚Äôs `scrollCallback`** + **`gsap.ticker`**, or **`ScrollTrigger`** on the track with **`scrub: 0`**.

Reduced motion ‚Äî `ScrollTrigger` writes progress from native `scrollTop`:

```ts
ScrollTrigger.create({
  trigger: track,
  scroller: shell,
  start: 'top top',
  end: () => `+=${Math.max(32, shell.clientHeight)}`,
  scrub: 0,
  invalidateOnRefresh: true,
  onUpdate(self) {
    shell.style.setProperty('--oa-cover-p', String(self.progress))
  },
})
```

Default path ‚Äî Lenis scroll value mapped to the same variable; Locomotive‚Äôs RAF is hooked to **GSAP‚Äôs ticker** so it lines up with other GSAP work:

```ts
const setCoverFromScroll = (scroll: number) => {
  const h = Math.max(32, shell.clientHeight)
  const p = Math.min(1, Math.max(0, scroll / h))
  shell.style.setProperty('--oa-cover-p', String(p))
}

const loco = new LocomotiveScroll({
  lenisOptions: {
    wrapper: shell,
    content: track,
    eventsTarget: shell,
    orientation: 'vertical',
    smoothWheel: true,
    lerp: 0.09,
    allowNestedScroll: true,
    autoResize: true,
  },
  scrollCallback: ({ scroll }) => {
    setCoverFromScroll(scroll)
  },
  initCustomTicker: (render) => {
    gsap.ticker.add(render)
  },
  destroyCustomTicker: (render) => {
    gsap.ticker.remove(render)
  },
})
```

The horizontal strip stays **`overflow-x: auto`**; the shell carries **`data-lenis-prevent`** so the site Lenis instance does not eat wheel events inside this embed.

## Part 4 ‚Äî Interior style rail: hover / focus swaps the panel (GSAP timeline)

**Target:** `.writing-oa-style__panel-inner` opacity and **`yPercent`**. **Driver:** **`onPointerEnter`** / **`onFocus`** per style row. **Runtime:** **`gsap.timeline`**: ease out ‚Üí **`call`** to swap `img.src` and caption ‚Üí **`fromTo`** in; **`gsap.killTweensOf(inner)`** before each new hover so stacked timelines do not fight.

From `WritingObjectArchiveInteriorStylePreview`:

```ts
gsap.killTweensOf(inner)

const tl = gsap.timeline({ defaults: { ease: 'power2.inOut' } })
tl.to(inner, { opacity: 0, yPercent: 5, duration: 0.14 }, 0)
tl.call(() => {
  writeDom(i)
})
tl.fromTo(
  inner,
  { yPercent: 9, opacity: 0.55 },
  { yPercent: 0, opacity: 1, duration: 0.42, ease: 'power2.out' },
)
```

Reduced motion: **`writeDom`** only and **`gsap.set(..., { clearProps: 'transform,opacity' })`**.

## Part 5 ‚Äî Resources: idle list, then hover / focus swaps the portrait (instant, fixed slot)

**Target:** Same **`img`** element in a **fixed 3:4** box; caption and overlay title text. **Driver:** hover/focus on list buttons. **Runtime:** **immediate DOM** updates‚Äîno GSAP on the image. **`ul { display: contents }`** lets each **`li`** stack with a centered **`.writing-oa-res__float`** (`z-index: 2`, **`pointer-events: none`**) while the **active** row stays above the art; **`.is-idle`** hides float and caption dock until the first pick.

Fixed slot in `_writing.scss`:

```scss
.writing-page__body .writing-oa-res__frame {
  width: min(16.5rem, 100%);
  max-width: 100%;
  aspect-ratio: 3 / 4;
  flex-shrink: 0;
  overflow: hidden;
}

.writing-page__body .writing-oa-res__frame img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  object-position: center;
}
```
